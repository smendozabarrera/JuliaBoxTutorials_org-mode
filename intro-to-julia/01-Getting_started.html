1.2 How to assign variables

All we need is a variable name, value, and an equal's sign! Julia will figure out types for us. 

my_answer = 42
typeof(my_answer)

42
Int64


my_pi = 3.14159
typeof(my_pi)

3.14159
Float64


😺 = "smiley cat!"
typeof(😺)

# \:smi + <tab> --> select with down arrow + <enter> ---> <tab> + <enter> to complete
